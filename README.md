# Multy Hub Loader

## How to Use

Paste this in your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/multyplyy/multyHubLoader/main/loader.luau"))()
```

If you're in an available game, it loads. If not, it tells you.

### Autoexec

You can also drop the loader into your executor's `autoexec` folder. Then it runs automatically every time you join any game. If the game is supported, the script loads by itself. If it's not, nothing happens. Pretty convenient if you play multiple supported games.

## "Is this a virus?"

No.

The loader script is right here in this repo — `loader.luau`. All it does is check what Roblox game you're playing and then loads the matching script. That's literally all it does.

The actual game scripts are obfuscated, which just means the code is scrambled so it can't be copied or read easily. That's standard for basically every script project out there. Obfuscation is not malware. It's just code protection.

Nothing here touches your files, installs anything, or does anything outside of Roblox.
